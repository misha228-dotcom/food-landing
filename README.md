# food-landing
Street Food Landing Page

Односторінковий лендинг для Street Food Merefa. Показує меню, переваги, контактну форму та посилання на соцмережі.

Особливості
Адаптивний дизайн
Секція Hero з головним зображенням і кнопкою “Buy Now”
Меню з фото, описом і цінами
Контактна форма
Посилання на Instagram, Facebook, TikTok, YouTube
Технології
HTML5
CSS3
Responsive design
Запуск

Відкрий index.html у браузері.
